# 🍔 Вкусный Выбор

**Вкусный Выбор** — Android-приложение для заказа еды с доставкой. Позволяет просматривать рестораны, изучать меню, добавлять блюда в корзину и оформлять заказы.

---

## Скриншоты

> _Скриншоты будут добавлены после сборки релизной версии._

---

## Функциональность

- **Главный экран** — карусель ресторанов с анимацией и параллакс-эффектом, история недавних заказов
- **Страница ресторана** — полное меню с разбивкой по категориям, вкладки «Меню» и «Акции»
- **Корзина** — добавление и удаление позиций, подсчёт итоговой суммы, бейдж с количеством товаров
- **Профиль** — базовый экран пользователя
- **Карта и Избранное** — подготовлены к реализации в следующих версиях

## Архитектура

Приложение построено по паттерну **MVVM**:

```
app/src/main/java/com/vkusnyvybor/
├── data/
│   ├── model/          # Модели данных (Restaurant, MenuItem, CartItem, Order…)
│   └── repository/     # MockRepository — источник данных
├── di/                 # Hilt-модули
└── ui/
    ├── components/     # Переиспользуемые Compose-компоненты
    ├── navigation/     # Маршруты и граф навигации
    ├── screens/        # Экраны: home, restaurant, cart, map, favorites, profile
    └── theme/          # Цвета, типографика, тема приложения
```

## Требования

- Android Studio Hedgehog или новее
- JDK 17
- Android SDK 34
- Минимальная версия Android: 8.0 (API 26)

---

## Поддерживаемые рестораны (демо-данные)

- **Вкусно — и точка**
- **Бургер Кинг**
- **Rostics**

Данные хранятся в `MockRepository` и легко заменяются на реальный API-бэкенд через уже подготовленный слой Retrofit.
